[EN]

1. Drop a Japanese Remember11 iso into the folder, rename it to "Remember11-jap.iso"

2. Run Remember11-patch.bat


Find useful links and latest patch versions on Github:
https://github.com/dreambottle/R11-psp-english

If you enjoyed the game on your PSP or emulator, give this project a star on Github!
